# DOTA 2 COUNTER PICKER

A web application built with React and Vite that helps Dota 2 players find the best counter picks against an enemy team. Select your opponent's heroes and get data-driven suggestions for heroes that have a high win rate against them.

## ‚ú® Features

- **Hero Browser**: Browse and search through all Dota 2 heroes.
- **Live Stats**: View up-to-date hero win rates and pick rates fetched from the OpenDota API.
- **Team Selection**: Select up to 5 enemy heroes to analyze.
- **Counter Analysis (In Progress)**: Get a list of suggested counter-picks based on the selected enemy team.
- **Modern UI**: Clean and responsive interface built with Tailwind CSS.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation & Running

1.  Clone the repository (you'll need to create one on GitHub first!).
    ```sh
    git clone https://github.com/your_username/dota-2-counter-picker.git
    ```
2.  Navigate to the project directory and install NPM packages.
    ```sh
    cd dota-2-counter-picker
    npm install
    ```
3.  Run the development server.
    ```sh
    npm run dev
    ```

Open http://localhost:5173 (or the address shown in your terminal) to view it in the browser.
